**Objective:** This repo is dedicated to recreating examples from [Make:Getting Started with Processing](https://www.amazon.com.au/Getting-Started-Processing-Casey-Reas/dp/1457187086).

**Motivation:** Apart from sending input/output to the console, C++ doesn’t support drawing graphics to a screen. Eventually you get bored seeing text and want to see colour and shapes. The solution is to use an external library (such as Raylib or SDL) or use Processing. 

**About Processing:** Think of Processing as C++ but with drawing functions. Whilst it’s based on Java, it shares a similar syntax to C++.

***NOTE:** Processing saves programs into a folder called a **Sketchbook.** The files in this repo are stored in a folder with the same name.*
